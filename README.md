# Iris Flower Classification

## Objective
Predict the species of an Iris flower using machine learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Algorithm
Random Forest Classifier

## Result
Achieved 100% accuracy on the test dataset.

## Dataset
Iris Dataset containing:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species
